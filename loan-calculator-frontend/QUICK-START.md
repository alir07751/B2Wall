# راهنمای شروع سریع

## نصب و اجرا (5 دقیقه)

### 1. نصب وابستگی‌ها

```bash
cd loan-calculator-frontend
npm install
```

### 2. اجرای برنامه

```bash
npm start
```

برنامه به صورت خودکار در مرورگر باز می‌شود: `http://localhost:3000`

### 3. تست با داده‌های نمونه

می‌توانید از داده‌های زیر برای تست استفاده کنید:

- **مبلغ وام**: 330,000,000
- **نرخ سود ماهیانه**: 4.5
- **پرداخت‌های اصل پول**:
  - روز 60: 30,000,000
  - روز 120: 50,000,000
  - روز 180: 100,000,000
  - روز 240: 150,000,000
- **تاریخ‌های پرداخت سود**: 180, 240

## ویژگی‌های رابط کاربری

### فرم ورودی

- ✅ ورودی مبلغ وام با فرمت خودکار (جداکننده هزارگان)
- ✅ ورودی نرخ سود با اعتبارسنجی
- ✅ اضافه/حذف سطرهای پرداخت اصل پول
- ✅ اضافه/حذف تاریخ‌های پرداخت سود
- ✅ بررسی خودکار مطابقت مجموع پرداخت‌ها با مبلغ وام
- ✅ پیام‌های خطا و هشدار به فارسی

### نمایش نتایج

- ✅ خلاصه اطلاعات در کارت‌های رنگی
- ✅ جدول تفصیلی با هایلایت روزهای پرداخت
- ✅ فرمت‌بندی خودکار ارز (تومان)
- ✅ طراحی واکنش‌گرا برای موبایل
- ✅ قابلیت چاپ گزارش

## نکات مهم

1. **مجموع پرداخت‌های اصل پول** باید دقیقاً برابر با **مبلغ وام** باشد
2. **روزها** باید اعداد مثبت باشند
3. در حالت توسعه، اگر اتصال به n8n برقرار نشود، محاسبه محلی انجام می‌شود
4. برای production، حتماً URL n8n webhook را در `.env` تنظیم کنید

## ساخت برای Production

```bash
npm run build
```

فایل‌های آماده در پوشه `build/` قرار می‌گیرند که می‌توانید آن‌ها را در هر سرور وب قرار دهید.
