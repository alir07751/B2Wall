# محاسبه‌گر بازپرداخت وام (Loan Repayment Calculator)

یک برنامه وب مدرن برای محاسبه برنامه بازپرداخت وام با پشتیبانی کامل از زبان فارسی و چیدمان راست به چپ (RTL).

## ویژگی‌ها

- ✅ رابط کاربری مدرن و واکنش‌گرا (Responsive)
- ✅ پشتیبانی کامل از راست به چپ (RTL) برای کاربران فارسی‌زبان
- ✅ فرم ورودی پویا برای پرداخت‌های اصل پول و تاریخ‌های پرداخت سود
- ✅ اعتبارسنجی لحظه‌ای و بازخورد بصری
- ✅ جدول تفصیلی بازپرداخت با فرمت‌بندی مناسب
- ✅ نمایش خلاصه اطلاعات و آمار کلیدی
- ✅ فرمت‌بندی خودکار ارز (تومان) با جداکننده‌های هزارگان

## نصب و راه‌اندازی

### پیش‌نیازها

- Node.js (نسخه 14 یا بالاتر)
- npm یا yarn

### مراحل نصب

1. نصب وابستگی‌ها:
```bash
npm install
```

2. تنظیم متغیر محیطی (اختیاری):
```bash
# ایجاد فایل .env
REACT_APP_N8N_WEBHOOK_URL=https://your-n8n-instance.com/webhook/loan-calculator
```

3. اجرای برنامه در حالت توسعه:
```bash
npm start
```

برنامه در آدرس `http://localhost:3000` در دسترس خواهد بود.

4. ساخت نسخه تولید:
```bash
npm run build
```

## استفاده

### ورودی‌های مورد نیاز

1. **مبلغ وام**: مبلغ کل وام به تومان
2. **نرخ سود ماهیانه**: درصد سود ماهیانه (مثال: 4.5)
3. **پرداخت‌های اصل پول**: آرایه‌ای از پرداخت‌ها با روز و مبلغ
4. **تاریخ‌های پرداخت سود**: روزهایی که سود انباشته پرداخت می‌شود

### خروجی

- **خلاصه**: آمار کلیدی شامل کل سود، کل بازپرداخت، مدت زمان
- **جدول تفصیلی**: برنامه روز به روز بازپرداخت با تمام جزئیات

## اتصال به n8n Backend

برای اتصال به n8n workflow:

1. URL webhook n8n خود را در فایل `.env` تنظیم کنید:
```
REACT_APP_N8N_WEBHOOK_URL=https://your-n8n-instance.com/webhook/loan-calculator
```

2. یا مستقیماً در فایل `src/utils/api.js` تغییر دهید.

در حالت توسعه، اگر اتصال به n8n برقرار نشود، برنامه از محاسبه محلی استفاده می‌کند.
